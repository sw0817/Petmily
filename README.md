# 차세대 반려동물 SNS PETMILY



## 필요 기능 정리 회의 내용(2021 01 11)



### 회원가입

- 사용자에게 많은 선택지를 주면 사용성이 낮아진다.

- 필수항목

  - 아이디(중복확인)
    - 빼도 될 듯 ?
  - 닉네임(중복확인)
  - 이메일(인증)
  - 생년월일
    - 날짜 파커 사용
  - 성별
  - 비밀번호(ex:8자 이상 + 특수문자 필)
  - 비밀번호 확인(여기까지 되어야 저장 버튼 활성화)

- 추가항목

  - 프로필 사진

  - 한 줄 자기소개

  - 반려동물 양육 여부(체크시 아래 활성화)

    - 몇 마리인지.
- 반려동물 이름
  - 반려동물 종류(강아지, 고양이, 기타)
  - 반려동물 사진

회원가입 및 로그인 즉 첫 페이지에서 이메일 입력창만 나오고, 회원 등록된 이메일이라면 로그인, 아니라면 회원가입 버튼이 나타나게 설정



### 피드

- 게시글 작성
  - 필수 항목
    - 사진 or 동영상
    - 글
  - 추가 항목
    - 태그
- 프로필 피드
  - 게시글 숨김 기능(나만 보기 - 내 프로필에서도 피드 변경 확인)
  - 좋아요
  - 댓글
- 태그 구독



### 다이어리

분야별 탭 기능(할지 말지 결정)

- 산책(어플 내 산책 시작 및 종료 기능 사용해도 좋음 - 모바일 전용(위치 켜졌을때만 가능))

  - 날씨(위 기능 쓴다면 날씨도 정보로 받아와서 자동)
  - 이동 경로

- 의료

- 피드와 다이어리의 차이점

  > 피드 : 전체공개(비공개게시물같은거 없는거로)
  > 댓글,  좋아요 기능 존재
  >
  > 다이어리 : 개인만 게시글을 올릴 수 있음
  > 개인이 공유한 사람들은 구경 가능
  > (댓글 좋아요는 넣던말던?)

  - 다이어리는 달력처럼 만들어 날짜별로 태그에 따라 알맞은 그림 활동으로 표시되고 들어가서 게시 다이어리 확인 가능
  - 뒷 배경사진 직접 올릴 수 있게(기본 이미지도 고를 수 있으면 좋을듯)



### 기타 목적 게시판(자유게시판, 공지사항)

- 사진 없어도 되는 글 작성

## 화면 설계

### 0. 각 화면 필요, 발생 데이터 및 페이지 이동 정리(2021 01 13)

1. 네비게이션(로그인시) - 다이어리, 마이페이지, 메뉴 목록, 메인페이지(로고)       

2. 완전초기화면(로그인 겸용) - 로그인 시도 시 (회원 이메일, 비밀번호) 검증

       - 회원가입

3. 회원가입 화면 - (회원 이메일, 닉네임 중복검사, 비밀번호 설정)

4. 메인화면(피드) - 각종 게시글 표시(다른 사람 정보(팔로잉 여부도), 검색태그, 특정 게시글(댓글), 댓글, 사진, 동영상, 글), `게시글 게시`, 갈 수 있는 링크 : 마이페이지(쪽지오면 표시), 다이어리, `세부메뉴(편의시설, 임시보호)`

5. 마이페이지 - 쪽지(갯수), 게시글 모음, 펫 등록(이름, 사진, 한 줄 소개, 종류, 성별), 개인정보 수정(로그아웃, 회원탈퇴), 펫 수정, 팔로잉 팔로워(사용자 링크)

6. 다이어리 - 전체(달력(활동 태그, 날씨)), 특정 날짜 보기(활동 태그, 날씨),  다이어리 작성(달력에 표시할수도), 뒷배경(자기가 꾸밀 수 있음)

7. 세부메뉴 - 편의시설, 임시보호

8. 편의시설 - 1) 검색을하고 2) 맵에 마커 + 왼쪽에 가게 이름과 주소 전화번호

   ​                - (여기서 애견 호텔 및 병원으로 임시보호 대체 가능성 있음)

10. 다른 사람 프로필 - 쪽지보내기, 팔로잉 기능(여부 + 하기), 게시글 보기, 펫 정보

13. 다이어리 작성 - 사진(not 필수), 글, 활동태그, 펫

### 1. 로그인 페이지

![](README.assets/firstLogin.gif)

첫 페이지는 로그인 페이지이다. "귀여운 동물친구들을 만나러 가볼까요 ?" 라는 문구에 트랜지션을 주어 동물친구들에 대한 관심을 끌 수 있도록 유도했다.

화면 상단에는 PETMILY 로고와 회원가입, 아이디/비밀번호 찾기 탭을 두었다.



![](README.assets/secondLogin.gif)

"우선 이메일을 입력해주세요" 라는 문구를 따라 이메일을 입력하면 비밀번호 입력 창이 나타나고, 요구하는 형식에 맞는 비밀번호 작성 시 로그인 버튼이 나타난다.

버튼의 색은 PETMILY의 대표 색이 될 보라색으로 선정하였다.



### 2. 회원가입 페이지

![](README.assets/loginToSignup.gif)

로그인 화면 상단 `Sign up` 버튼을 클릭하면 회원가입 페이지로 이동한다.

페이지 이동 후, 차례대로 로고, 가입 독려 문구, 회원가입 폼이 나타난다.



![](README.assets/signup.gif)

가입에는 이메일(아이디), 닉네임, 비밀번호, 이름, 생년월일, 성별이 필요하고, 각 항목에 대해 유효하지 않은 입력 시 유효한 방식으로 입력해달라는 문구가 각 항목 아래에 나타나도록 하였다.

로그인과 마찬가지로, 모든 항목에 유효한 입력을 완료할 시 회원가입 버튼이 나타난다.
